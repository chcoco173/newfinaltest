
React 애플리케이션을 생성(TypeScript 템플릿을 사용)
> create-react-app . --template typescript

TypeScript를 전역적으로 설치(TypeScript 컴파일러를 사용)
> npm install -g typescript

ts-node를 전역적으로 설치(TypeScript 파일을 직접 실행)
> npm install -g ts-node

Node.js를 사용하여 특정 JavaScript 파일을 실행
> node src/0_basic/0_notypescript.js



* 자바스크립트 파일
  - js --> jsx (리액트에서 권장)

* 타입스크립트 파일  
  - ts --> tsx (리액트에서 권장)
